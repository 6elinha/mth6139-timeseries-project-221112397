# Time Series Forecasting with Google Trends Data

This project analyses and forecasts search interest using Google Trends data across multiple food and lifestyle-related topics. The goal is to identify trends, seasonality, and patterns in public interest over time, and to explore how these insights can inform real-world decision-making.

## 📊 Overview

Using R and the Prophet forecasting model developed by Meta, this project examines three years of Google Trends data for the following search terms:

- Banana Bread  
- Lemon Drizzle  
- Gingerbread  
- Tiramisu  
- Protein  
- Vegan  

Each dataset is analysed to identify:
- Long-term trends  
- Seasonal patterns  
- Forecasted future interest  

## 🛠 Tools & Technologies

- R
- RMarkdown
- Prophet (Meta)
- Google Trends data
- GitHub Pages (for deployment)

## 📈 Key Objectives

- Clean and preprocess time series data  
- Apply forecasting models using Prophet  
- Visualise trends and seasonality  
- Compare different search terms  
- Interpret results from a business perspective

- Some products show strong seasonal patterns (e.g. gingerbread in winter)  
- Others show consistent or growing demand (e.g. banana bread, protein)  
- Lifestyle-related trends such as veganism and protein consumption show different patterns compared to traditional bakery items
